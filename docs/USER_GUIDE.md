## 系统介绍

Flight Management System 是一个专业的航班管理系统前端应用程序，为用户提供直观、高效的航班管理体验。系统具有以下特点：

- **现代化界面**: 采用扁平化设计，支持明暗主题切换
- **功能完整**: 涵盖航班查询、预订、用户管理等核心功能
- **操作简便**: 直观的操作流程，易于上手
- **数据安全**: 本地数据存储，保护用户隐私
- **响应迅速**: 优化的性能，流畅的用户体验

## 安装和启动

### 系统要求

- **操作系统**: Windows 10+, macOS 10.14+, Linux (Ubuntu 18.04+)
- **内存**: 最少 4GB RAM
- **存储空间**: 至少 500MB 可用空间
- **显示器**: 分辨率不低于 1024x768

### 安装步骤

#### Windows
1. 下载 `FlightSystem-Setup.exe` 安装程序
2. 双击运行安装程序
3. 按照安装向导完成安装
4. 从开始菜单启动应用程序

#### macOS
1. 下载 `FlightSystem.dmg` 磁盘映像
2. 双击打开磁盘映像
3. 将应用程序拖拽到应用程序文件夹
4. 从 Launchpad 启动应用程序

#### Linux
```bash
# 使用包管理器安装 (Ubuntu/Debian)
sudo apt-get install flightsystem

# 或下载 AppImage
chmod +x FlightSystem.AppImage
./FlightSystem.AppImage
```

### 首次启动

1. 双击应用程序图标启动
2. 系统将显示欢迎界面
3. 选择界面主题（明色/暗色）
4. 进入主界面

## 航班查询

### 基本搜索

1. **进入查询页面**
   - 点击工具栏的"航班查询"按钮
   - 或使用快捷键 `Ctrl+S`

2. **填写搜索条件**
   ```
   出发地: 北京          [必填]
   目的地: 上海          [必填]
   航班类型: 单程        [可选]
   出发日期: 2024-01-15  [必填]
   返程日期: 2024-01-22  [往返时必填]
   乘客人数: 1           [可选]
   舱位等级: 经济舱      [可选]
   ```

3. **执行搜索**
   - 点击"搜索航班"按钮
   - 系统将显示符合条件的航班列表

### 高级搜索技巧

#### 模糊搜索
- 输入城市名的一部分，如"北"可以匹配"北京"
- 支持拼音搜索，如"beijing"可以匹配"北京"

#### 日期范围
- 点击日期选择器选择具体日期
- 使用日历界面快速浏览日期
- 支持键盘输入日期格式

#### 多条件组合
- 可以组合多个条件进行精确搜索
- 系统会实时显示匹配结果数量

### 搜索结果

#### 结果展示
```
航班号    航空公司      出发地  目的地  出发时间  到达时间  价格    状态
CA1234    中国国际航空  北京    上海    08:00     10:30     ¥1,280  准点
MU5678    东方航空      北京    上海    09:15     11:45     ¥1,150  准点
```

#### 状态说明
- **准点**: 航班按计划时间运行
- **延误**: 航班延迟起飞（红色标识）
- **取消**: 航班已取消（红色标识）

#### 结果操作
- **查看详情**: 点击航班行查看详细信息
- **排序**: 点击列标题进行排序
- **筛选**: 使用状态筛选器过滤结果

## 航班预订

### 预订流程

#### 第一步：选择航班
1. 进入航班预订页面
2. 在航班列表中选择目标航班
3. 点击"查看详情"确认航班信息

#### 第二步：填写乘客信息
```
基本信息:
- 姓名: 张三
- 身份证号: 110101199001011234
- 性别: 男
- 出生日期: 1990-01-01

联系方式:
- 手机号码: 13800138000
- 邮箱地址: zhangsan@example.com

乘客人数:
- 成人: 1人
- 儿童: 0人
- 婴儿: 0人
```

#### 第三步：选择服务
```
附加服务:
□ 航班保险 (¥200)
□ 特殊餐食 (¥50)
□ 优选座位 (¥100)
```

#### 第四步：支付信息
```
支付方式:
[信用卡] [借记卡] [支付宝] [微信支付] [银行转账]

卡片信息: (信用卡/借记卡)
- 卡号: ****-****-****-1234
- 持卡人: 张三
- 有效期: 12/25
- CVV: ***
```

#### 第五步：确认预订
1. 检查预订摘要信息
2. 确认所有信息无误
3. 点击"确认预订"按钮
4. 系统显示预订成功信息

### 预订修改

#### 修改乘客信息
1. 在用户管理页面找到预订记录
2. 点击"编辑"按钮
3. 修改相关信息
4. 保存更改

#### 取消预订
1. 找到要取消的预订
2. 点击"取消预订"按钮
3. 选择取消原因
4. 确认取消操作

### 价格计算

系统会自动计算总价：
```
基础价格: ¥1,280 × 1成人 = ¥1,280
税费: ¥128 (10%)
保险费: ¥200 (可选)
餐食费: ¥50 (可选)
座位费: ¥100 (可选)

总计: ¥1,658
```

## 用户管理

### 用户账户

#### 创建新用户
1. 点击"添加用户"按钮
2. 填写用户基本信息
3. 设置用户角色
4. 保存用户信息

```
用户信息:
- 用户名: newuser
- 密码: ********
- 邮箱: user@example.com
- 电话: 13800138000
- 姓名: 张三
- 角色: 普通用户
- 状态: 活跃
```

#### 用户角色说明
- **普通用户**: 基本查询和预订权限
- **VIP用户**: 增值服务权限
- **客服**: 客户服务权限
- **管理员**: 系统管理权限

#### 用户状态管理
- **活跃**: 用户可正常使用系统
- **暂停**: 临时限制用户权限
- **禁用**: 完全禁止用户访问

### 用户搜索

#### 基本搜索
- 输入用户名、邮箱或电话号码
- 选择搜索类型
- 点击"搜索"按钮

#### 高级筛选
- 按用户角色筛选
- 按用户状态筛选
- 按注册时间筛选

### 批量操作

#### 批量导入
1. 准备CSV格式的用户数据
2. 点击"导入用户"按钮
3. 选择文件并上传
4. 确认导入结果

#### 批量导出
1. 设置筛选条件
2. 点击"导出数据"按钮
3. 选择导出格式
4. 保存导出文件

## 常见问题

### Q: 忘记密码怎么办？
A: 联系系统管理员重置密码，或使用"忘记密码"功能自助重置。

### Q: 航班搜索没有结果？
A: 检查搜索条件是否正确，尝试使用更宽泛的搜索条件。

### Q: 预订失败怎么办？
A: 检查网络连接，确认支付信息正确，或联系客服协助。

### Q: 如何导出数据？
A: 在相应页面点击"导出"按钮，选择导出格式和保存位置。

### Q: 系统运行缓慢？
A: 检查系统资源使用情况，关闭不必要的应用程序，重启系统。

### Q: 如何联系技术支持？
A: 发送邮件至 support@flightsystems.com 或致电 400-123-4567。
