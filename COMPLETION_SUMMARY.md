## 🔧 技术栈详解

### 前端框架
```
Qt 5.15+ / Qt 6.x
├── 核心库: Core, GUI, Network
├── HTTP处理: QNetworkAccessManager
├── JSON处理: QJsonDocument, QJsonObject
├── 数据存储: QSettings
└── 线程: QThread (如需)
```

### API架构
```
前端 (Qt) → HTTP/HTTPS → 后端 API
├── 认证: JWT Token
├── 数据格式: JSON
├── 错误处理: 统一格式
└── 版本管理: v1
```

### 编程语言
```
C++17
├── 现代特性: auto, lambda, smart_ptr
├── STL容器: QList, QMap, QVector
├── 内存管理: 智能指针
└── 异常处理: try-catch
```

## 🚀 部署指南

### 环境要求
```
操作系统: Windows/macOS/Linux
Qt版本: 5.15+ 或 6.x
编译器: GCC/MSVC/Clang
内存: 512MB+
磁盘: 100MB+
```

### 编译步骤
```bash
# 1. 配置
qmake FlightSystem.pro

# 2. 编译
make

# 3. 运行
./FlightSystem
```

### Mock API 部署

**Node.js**
```bash
npm install
npm start
# 访问 http://localhost:3000
```

**Python**
```bash
pip install -r requirements.txt
python app.py
# 访问 http://localhost:3000
```

**Go**
```bash
go run main.go
# 访问 http://localhost:3000
```
